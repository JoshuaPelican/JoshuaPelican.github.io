Trouble in Tornado Town

  • Team of 10
  • Unity C#
  • ~2 Years

Trouble in Tornado Town is an physics-based destruction game featuring a giant tornado! TTT was produced by a team of 10 college students outside of class time to gain experience shipping a game. I was the lead programmer and created the wacky gameplay systems that make it so fun to play! I optimized the physics and rendering pipelines to work for the thousands of object we wanted on-screen. It is now fully released on Steam following 2 major updates.

  • Designed & programmed core gameplay loop
  • Complex tornado simulations
  • Optimized for 3000+ physics objects
  • Shipped on steam with multiple updates

Desert Darts

  • Solo
  • Unity C#
  • ~2 Days

Desert Darts is an endless arcade game and my first mobile project. Using the devices gyroscopic information, the player controls a target with the goal of landing precice hits in succession. Desert Darts features an adaptive difficulty system and complex dart pattern generation for engaging and replayable gameplay. Desert Darts was also created with the aim of an accessable icon-only interface. I was the sole creator of this project.

  • Adaptive difficulty
  • Procedural dart patterns
  • Android compatability
  • Accessible, icon-only interface
